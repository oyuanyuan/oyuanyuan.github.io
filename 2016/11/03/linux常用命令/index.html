linux常用命令

发表于 2016-11-03   |     |   阅读次数
ls

显示文件或目录

ls -l  显示文件或目录详细信息
ls -a   显示所有文件或目录（包括隐藏文件）

cd

切换目录

touch

创建文件

mkdir

创建目录
mkdir dir  创建目录dir
mkdir -p dir/dir1   创建多级目录，若父目录不存在，会自动创建

cp

拷贝文件或目录

1.拷贝文件
cp file1 file2  拷贝文件file1，拷贝的新文件命名为 file2
cp dir file1  拷贝文件file1到目录dir下面

2.拷贝目录
cp -r dir1 dir2  如果目录dir2不存在，则创建目录dir2，并拷贝目录dir1下所有文件到目录dir2下；如果目录dir2存在，则拷贝整个dir1目录到dir2下
cp -a dir1 dir2  和-r一样，只是-a会把链接、文件属性一起拷贝，-r则不会
